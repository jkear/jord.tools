# MLX

## Quick Overview
MLX is an array framework for machine learning on Apple silicon, brought to you by Apple machine learning research. It features a Python API that closely follows NumPy and PyTorch, making it easy to build efficient models for Mac.

## My Experience
MLX is my go-to framework for running local LLMs and training small models directly on my MacBook Pro. The performance gains on Apple Silicon are significant.

## Quick Start
```bash
# Installation command
pip install mlx
```
