# n8n

## Quick Overview
n8n is an extendable workflow automation tool. With a fair-code distribution model, n8n will always have visible source code, be available to self-host, and allow you to add your own custom functions, logic and apps.

## My Experience
I use n8n for connecting various services and automating background tasks. Its recent AI agent nodes have made it a powerful tool for building autonomous workflows.

## Quick Start
```bash
# Installation command
npm install n8n -g
n8n start
```
