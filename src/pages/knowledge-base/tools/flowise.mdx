# Flowise

## Quick Overview
Flowise is an open source low-code tool for developers to build customized LLM orchestration flows & AI agents. It provides a drag-and-drop interface to visualize and build LangChain apps.

## My Experience
Flowise is excellent for rapid prototyping of agent flows before committing to code. I use it to visualize complex chains and test different component combinations.

## Quick Start
```bash
# Installation command
npm install -g flowise
npx flowise start
```
