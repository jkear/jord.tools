# Marker

## Quick Overview
Marker is a tool that converts PDF, EPUB, and MOBI to markdown. It's designed to be fast and accurate, handling equations, tables, and code blocks better than many other converters.

## My Experience
Marker is essential for my RAG pipelines. Converting documents to clean markdown is the first step in creating high-quality embeddings for knowledge retrieval.

## Quick Start
```bash
# Installation command
pip install marker-pdf
```
